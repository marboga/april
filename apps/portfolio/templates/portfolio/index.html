Capture!more_vert
Capture!

Built for iOS, Capture! is a real-life multiplayer capture the flag game.

Users can create or join games, defining boundaries for the play area. Flags are randomly generated within a range of the far boundary corner.

Players can tag opponents or be tagged, depending on which team's zone the players are in, with sound notifications to both players. Also, the app beeps at users who are out of bounds once the game has started.

The real-time updates of other users' positions on the map are facilitated by Firebase.

  • iOS
  • Swift
  • Firebase
  • CoreLocation
  • MapKit
  • CocoaPods

Sprog

Built in full MEAN, Sprog is a fully featured flashcard app for learning languages.

Users can select the languages they wish to learn, browse existing flashcards and easily create new ones using Google Translate and image search and auto-fill.

Users can also quiz themselves on custom-built decks, clicking on the card to hear a native speaker say the word aloud.

  • Node.js
  • Angular
  • Twitter Bootstrap
  • Google Translate API
  • Forvo API
